# Bible Companion
A comprehensive and interactive Bible study experience, featuring hot switching between translations and commentaries, note-taking capabilities, and reflection checklists.

### Features

* **Translation Switching**: Quickly switch between different translations of the Bible (e.g., BSB, ASV, KJV) to suit your reading preferences.
* **Commentary Support**: Access various biblical commentaries for in-depth understanding and insights into specific passages.
* **Note-taking**: Take notes on each chapter, allowing you to record thoughts, questions, or observations as you study the Bible.
* **Reflection Checklists**: Use pre-designed reflection checklists (or make your own) to guide your personal reflections and devotions as you read the Bible.

### Getting Started with Flutter
If you're new to Flutter, follow these steps to set up a development environment:

1. **Install Flutter**: Go to <https://flutter.dev/docs/get-started/install> and follow the installation instructions for your operating system.
2. **Create a New Project**: Open a terminal or command prompt and run `flutter create my_bible_app` (replace "my_bible_app" with your desired app name).
3. **Open the Project Directory**: Navigate to the project directory using your preferred code editor or IDE.
4. **Run the App**: Run `flutter run` in the terminal or command prompt to start the emulator and test the app.
